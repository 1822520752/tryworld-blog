## 📖 项目简介

**试界 TryWorld** 是一个现代化的个人博客网站，采用前后端分离架构开发。网站设计风格独特，融合了赛博朋克元素与霓虹灯效果，为访客带来沉浸式的浏览体验。

作为一个高中生 AI 开发者，我通过这个博客分享我的：
- 🎯 技术学习心得
- 💻 项目开发经验  
- 🏆 个人成长历程
- 🔬 AI 探索实践

## 🚀 快速开始

### 📋 环境要求

在开始之前，请确保你的开发环境满足以下要求：

| 环境 | 最低版本 | 推荐版本 |
|:---:|:---:|:---:|
| Node.js | 16.0.0 | 18.x 或更高 |
| npm | 8.0.0 | 9.x 或更高 |
| Git | 2.0.0 | 最新版本 |

### 📥 克隆项目

```bash
# 克隆仓库
git clone https://github.com/1822520752/tryworld-blog.git

# 进入项目目录
cd tryworld-blog
```

### 📦 安装依赖

```bash
# 安装前端依赖
cd frontend
npm install

# 安装后端依赖
cd ../backend
npm install
```

### 🔧 启动项目

#### 开发模式

```bash
# 终端 1：启动后端服务
cd backend
npm start
# 后端服务运行在 http://localhost:47191

# 终端 2：启动前端开发服务器
cd frontend
npm run dev
# 前端服务运行在 http://localhost:5173
```

#### 生产构建

```bash
# 构建前端生产版本
cd frontend
npm run build

# 预览构建结果
npm run preview
```

### 🌐 访问地址

| 服务 | 地址 | 说明 |
|:---:|:---:|:---|
| 前端开发服务器 | http://localhost:5173 | Vite 开发服务器 |
| 后端 API 服务 | http://localhost:47191 | Express 服务器 |

## 🎨 自定义配置

### 修改个人信息

编辑 `frontend/src/components/ProfileCard.jsx` 文件：

```javascript
// 修改头像
src="你的头像URL"

// 修改姓名和职位
<span className="name-text">你的名字</span>
<span className="role-text">你的职位</span>

// 修改个人简介
const bio = "你的个人简介...";
```

### 修改项目展示

编辑 `frontend/src/components/ProjectsSection.jsx` 文件，添加你的项目信息。

### 修改主题颜色

编辑 `frontend/src/styles/global.css` 文件：

```css
:root {
  --primary-color: #00ff88;    /* 主色调 */
  --secondary-color: #ff00ff;  /* 次要色调 */
  --accent-color: #00ffff;     /* 强调色 */
}
```
